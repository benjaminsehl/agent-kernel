# References

This repository is a downstream synthesis of several ideas.

## agent-kernel
- Website: https://agent-kernel.dev/
- GitHub: https://github.com/oguzbilgic/agent-kernel

Why it matters:
- established the small-markdown-kernel pattern
- showed how `AGENTS.md` can bootstrap stateful agent behavior
- provided the base repo shape that inspired this project

## LLM Wiki by Andrej Karpathy
- Gist: https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f

Why it matters:
- sharpened the distinction between raw source material and compiled knowledge
- emphasized that good answers should be filed back into the knowledge artifact
- framed the markdown corpus as a persistent, compounding asset

## OpenClaw dreaming
- Docs: https://docs.openclaw.ai/concepts/dreaming
- Alternate docs path: https://docs.openclaw.ai/concepts/memory-dreaming

Why it matters:
- introduced a practical model for memory consolidation and promotion discipline
- inspired the reflection/dreaming layer in this repo
- emphasized reviewable promotion from short-term signals to durable memory
